# Projet Flask – Jeu Mastermind 
Ce projet a été réalisé au cours du second semestre de mon Master 1, dans le cadre du cours de Programmation Orientée Objet.

L’objectif du projet était de concevoir et d’implémenter une application logicielle basée sur le jeu Mastermind, en appliquant les principes fondamentaux de la programmation orientée objet.
Le projet a été développé en Python et repose sur le framework Flask afin de proposer une interface web interactive.

Ce travail m’a permis de mettre en pratique la structuration du code, la conception de classes, la gestion de la logique métier ainsi que l’interaction entre le backend et l’interface utilisateur.
