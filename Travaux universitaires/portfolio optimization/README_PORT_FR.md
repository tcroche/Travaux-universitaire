# Projet Python – Analyse d’ETFs et allocations anonymisées

Ce projet a été réalisé dans le cadre d’un projet Python de finance quantitative.
Il consiste à analyser un univers de 105 ETFs anonymisés, 14 grandes classes d’actifs, ainsi que deux portefeuilles anonymisés (« Mystery Allocations ») construits à partir de ces ETFs.

L’objectif du projet est triple :
- analyser les profils de risque et de performance des ETFs ;
- étudier les relations entre ETFs et grandes classes d’actifs ;
- identifier et caractériser la composition et le profil de risque des deux portefeuilles anonymisés.

Les résultats montrent que :
- l’univers d’ETFs couvre un large spectre de profils de risque, allant d’expositions obligataires défensives à des actifs plus volatils ;
- la première allocation anonymisée est très bien expliquée par une combinaison statique d’ETFs, avec une exposition majoritairement orientée vers les actions ;
- la seconde allocation présente une dynamique plus évolutive, avec une exposition plus diversifiée et légèrement plus cyclique.
